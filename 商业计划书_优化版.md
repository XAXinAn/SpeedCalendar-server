# SpeedCalendar 商业计划书

## 1. 执行摘要 (Executive Summary)

**SpeedCalendar** 是一款基于 Android 平台的下一代智能日程管理应用，旨在通过人工智能（AI）和光学字符识别（OCR）技术，彻底改变用户管理时间的方式。在信息爆炸的时代，传统的日历应用往往功能单一，无法高效处理来自聊天记录、截图或文档中的碎片化日程信息。SpeedCalendar 通过创新的“悬浮窗 OCR + AI 语义分析”工作流，实现了从屏幕信息到日程安排的“一键直达”，并结合群组协作功能，为个人和团队提供了一站式的智能时间管理解决方案。

本项目采用前沿的 Jetpack Compose 技术栈构建原生 Android 客户端，后端基于稳健的 Spring Boot 架构，确保了系统的高性能、高可用性和数据安全性。

## 2. 痛点与解决方案 (Problem & Solution)

### 2.1 痛点分析
- **信息碎片化**：日程信息散落在微信聊天、邮件、备忘录和图片截图中，手动整理耗时费力。
- **操作繁琐**：传统日历添加日程需要手动填写标题、时间、地点等多个字段，步骤繁琐，用户体验差。
- **协作困难**：团队日程协调往往需要反复沟通确认，缺乏高效的共享和同步机制。
- **隐私顾虑**：用户对于个人日程数据的隐私安全日益关注，需要更精细的权限控制。

### 2.2 解决方案：SpeedCalendar
- **AI 智能助手**：内置多会话 AI 聊天助手，支持自然语言交互，自动提取文本中的日程信息。
- **悬浮窗 OCR 黑科技**：独创的全局悬浮窗功能，支持前台截屏 OCR 识别。用户无需离开当前应用（如微信、钉钉），即可通过“截图-识别-AI分析”的工作流，在几秒钟内将屏幕信息转化为结构化日程。
- **无缝群组协作**：支持创建和管理群组，实现团队日程的实时共享与同步，提升协作效率。
- **隐私优先设计**：提供精细的隐私设置（公开、仅好友可见、私密），保障用户数据安全。

## 3. 产品与技术亮点 (Product & Technology)

### 3.1 核心功能模块

#### **智能日历 (Smart Calendar)**
- **交互体验**：基于 Jetpack Compose 打造的丝滑月视图，支持手势滑动切换月份，拖拽操作流畅自然。
- **便捷管理**：悬浮按钮（FAB）快速添加日程，内置自定义的年/月选择器、时间选择器和群组选择器。
- **详情交互**：日程列表支持侧滑编辑/删除，采用 BottomSheet 底部弹窗设计，操作符合现代移动端习惯。

#### **AI 智能助手 (AI Assistant)**
- **多模态交互**：支持多会话聊天管理，侧边栏（Drawer）查看历史记录。
- **OCR 快速录入**：集成 PaddleOCR 引擎，支持通过相机拍摄或相册导入图片，自动识别文字并喂给 AI 进行日程分析。
- **长文本处理**：优化的长文本选择容器，方便用户复制和处理复杂信息。

#### **全局悬浮窗 OCR (Floating Window OCR)**
- **技术实现**：基于 `MediaProjection` 服务和 `FloatingWindowService`，实现应用层之上的全局悬浮窗。
- **工作流**：
    1. **悬浮球常驻**：可拖拽、自动贴边，不干扰正常操作。
    2. **一键截屏**：点击悬浮球触发截屏，进入 `ScreenCropActivity` 进行区域裁剪。
    3. **本地识别**：调用本地 PaddleOCR 模型进行毫秒级文字识别。
    4. **AI 分析**：识别结果自动发送至 AI 助手，AI 智能解析时间、地点、事件，并生成日程建议。
    5. **即时反馈**：悬浮窗展开面板实时显示识别文本和 AI 回复，用户可直接确认添加。

#### **群组与协作 (Group Collaboration)**
- **全流程管理**：支持群组创建、搜索加入、成员管理（踢人、权限控制）。
- **权限体系**：基于角色的权限控制（管理员/普通成员），支持邀请码机制。

#### **个人中心与隐私 (Mine & Settings)**
- **安全认证**：手机号验证码登录，登录页（LoginSheet）集成倒计时和协议签署流程。
- **隐私配置**：支持针对不同对象的日程可见性设置（PrivacySettingsScreen），采用乐观更新（Optimistic UI）策略，提升设置体验。

### 3.2 技术架构 (Technical Architecture)

#### **前端架构 (Android Client)**
- **开发语言**：Kotlin
- **UI 框架**：Jetpack Compose (Material3 Design)
- **架构模式**：MVVM (Model-View-ViewModel) + Unidirectional Data Flow (UDF)
- **核心组件**：
    - **Navigation Compose**：管理单 Activity 多 Screen 的页面跳转。
    - **ViewModel + StateFlow**：实现响应式数据流和 UI 状态管理。
    - **Coroutines**：处理异步任务，保证主线程流畅。
    - **Retrofit + OkHttp**：网络请求层，封装了 `AuthInterceptor` 实现双 Token（Access/Refresh）自动刷新机制，无感处理 401 状态。
    - **PaddleOCR4Android**：端侧 OCR 推理引擎，离线识别，保护隐私且速度快。
    - **MediaProjection Service**：前台服务，确保持续的屏幕采集能力。
    - **Coil**：高性能图片加载库。

#### **后端架构 (Server)**
- **开发框架**：Spring Boot
- **API 设计**：RESTful API 规范
- **数据存储**：MySQL
- **安全机制**：
    - **JWT (JSON Web Token)**：实现无状态认证。
    - **双 Token 机制**：Access Token (2小时有效期) + Refresh Token (30天有效期)，平衡安全性与用户体验。
- **核心服务**：
    - **Auth Service**：处理注册、登录、验证码发送。
    - **Schedule Service**：日程 CRUD 业务逻辑。
    - **AI Chat Service**：对接大模型接口，处理对话上下文。
    - **Group Service**：群组关系与权限管理。

## 4. 市场分析 (Market Analysis)

### 4.1 目标用户
- **高校学生**：课程表、社团活动、考试提醒繁多，习惯使用移动端工具。
- **职场人士**：会议安排、项目节点密集，需要高效的日程整理工具。
- **团队管理者**：需要统筹团队日程，进行任务分发和进度同步。

### 4.2 竞争优势
- **技术壁垒**：端侧 OCR + 悬浮窗交互构建了独特的数据采集入口，相比竞品（通常需要手动复制粘贴）效率提升 300% 以上。
- **体验优势**：Jetpack Compose 带来的现代化流畅 UI，以及 AI 的深度整合，提供了超越传统日历的智能化体验。
- **隐私安全**：本地 OCR 识别减少了敏感图片上传云端的风险。
