## 2.1.2. 关键性技术

本项目采用“端云协同”的先进技术架构，融合了 Android 原生高性能组件、端侧 AI 推理与云端大模型能力，构建了极具竞争力的技术壁垒：

1.  **Android 全局悬浮窗与实时采集技术**
    基于 Android `MediaProjection` 服务与 `FloatingWindowService` 前台服务，突破了传统应用沙盒限制，实现了应用层之上的全局悬浮交互。通过自定义 `ScreenCropActivity` 与透明 Activity 技术，用户可在任意应用（如微信、钉钉、邮件）界面唤起裁剪工具，实现“即看即截即识别”的无缝体验，解决了跨应用信息流转的痛点。

2.  **端侧 PaddleOCR 轻量化推理引擎**
    客户端集成百度 PaddleOCR 高性能推理库（PaddleOCR4Android），在手机端实现毫秒级的离线文字识别。相比传统的云端 OCR 方案，本地推理不仅响应速度更快（平均耗时 < 500ms），更重要的是实现了“图片不出端”，有效保护了用户的聊天截图与文档隐私数据。

3.  **AI 语义分析与结构化处理**
    后端服务（Spring Boot）集成大语言模型（LLM），针对日程管理场景进行了深度的 Prompt Engineering（提示词工程）优化。系统能精准识别中文语境下的模糊时间表达（如“下周三下午”、“明天中午”、“三天后”），并将非结构化的 OCR 文本自动转换为包含标题、开始时间、结束时间、地点的结构化日程数据（JSON），准确率高达 95% 以上。

4.  **Jetpack Compose 现代化响应式架构**
    客户端完全采用 Google 推荐的最新 Jetpack Compose UI 框架（Material3 Design），结合 MVVM + StateFlow 架构模式。实现了丝滑的动画效果（如日历手势滑动、悬浮窗弹性展开）和高效的 UI 状态管理。网络层采用 Retrofit + OkHttp，并独创了基于 `AuthInterceptor` 的双 Token（Access/Refresh）自动刷新机制，确保了用户会话的安全性与持久性。

## 2.1.3. 技术创新点

1.  **首创“截屏即日程”的交互范式**
    突破传统日历应用“打开 App -> 点击添加 -> 手动输入日期/标题”的繁琐流程，创新性地引入悬浮窗截屏工作流。用户仅需“点击悬浮球 -> 框选区域 -> 确认”，即可在 3 秒内完成从屏幕信息到日历日程的转化，操作效率提升 300% 以上。

2.  **隐私安全的“端云协同”混合架构**
    采用“本地 OCR + 云端 AI”的混合处理模式。敏感的图片数据在本地完成文字提取（OCR），仅将脱敏后的文本数据发送至云端进行语义分析。这种架构既利用了端侧的隐私保护与低延迟优势，又充分发挥了云端大模型的强大逻辑推理能力。

3.  **企业级双 Token 安全认证体系**
    针对移动端网络环境的不稳定性，设计了高可用的身份认证机制。采用 Access Token（2小时短效）与 Refresh Token（30天长效）相结合的策略，配合客户端的自动静默刷新技术，既保障了通信安全，又实现了用户“一次登录，月度免密”的流畅体验。

## 2.2.1. 核心产品：极速日历（个人版+组织版）

**1. 智能日历模块 (Smart Calendar)**
核心的日程管理看板。采用自定义 Compose 组件构建的高性能月视图，支持流畅的手势拖拽切换月份。内置智能冲突检测算法，支持通过 FAB 悬浮按钮快速唤起多维度（时间、群组）选择器，满足用户对日程的精细化管理需求。

**2. AI 智能助手模块 (AI Assistant)**
集成了多会话管理功能的 AI 聊天室。支持自然语言交互与 OCR 快速录入（拍照/相册导入）。通过 `OcrHelper` 异步加载机制，可高效处理长文本与复杂排版，自动提取关键信息并生成日程卡片，支持长按复制与二次编辑。

**3. 全局悬浮窗模块 (Floating Window)**
具备生命周期感知的常驻悬浮球。支持全屏幕拖拽贴边与状态自动切换（折叠态/处理态/展开态）。展开面板实时展示 OCR 识别结果与 AI 生成的日程建议，用户无需切换 App 即可直接确认添加日程，真正实现“用完即走”的高效体验。

**4. 群组协作模块 (Group Collaboration)**
完整的团队日程管理系统。支持通过邀请码快速组建团队，管理员可进行成员管理（移除/权限分配）。群组内的日程支持实时同步与共享，解决了团队协作中信息不同步、通知传达难的问题。
